# On the Rocks Android

Native Android port scaffold for the On the Rocks cocktail app.

## Open in Android Studio

1. Open Android Studio.
2. Choose **Open**.
3. Select this folder: `OnTheRocksAndroid`.
4. Let Gradle sync finish.
5. Run the `app` configuration on an emulator or connected Android device.

The current app is a Compose starter screen. The next porting step is moving over the cocktail catalog and rebuilding the Discover screen.
